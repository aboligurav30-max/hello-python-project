# Week 1 - Personal Introduction Program

## Project Overview

This is a beginner Python project that collects user information and displays a personalized welcome message.

## Objectives

- Learn Python basics
- Understand variables
- Use input() and print()
- Practice f-strings
- Build confidence in writing simple programs

---

## Features

- Takes user input
- Stores data in variables
- Displays a formatted welcome message
- Uses f-strings
- Beginner-friendly code

---

## Technologies Used

- Python 3.x
- Visual Studio Code

## How to Run

Open terminal inside project folder.

Run:

```bash
python personal_intro.py
```
